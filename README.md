<h2 id="project">Projeto</h2>
<p>O projeto Food Explorer é uma aplicação de cardápio digital que trata-se do desafio final do curso Explorer da Rocketseat.</p>

<p>Neste repositório está disponível o back-end do projeto.</p>

<h2 id="pages">Páginas</h2>
1. Usuários
2. Pratos
3. Ingredientes dos pratos
4. Favoritos
5. Carrinhos
6. Itens dos carrinhos

<h2 id="technologies">Tecnologias</h2>
1. Bcrypt.js
2. Node.js
3. Express.js
4. Knex.js
5. SQLite
6. SQLite3
7. express-async-errors
8. JSON Web Token
9. Multer
10. PM2
11. CORS

<h2 id="usage">Para executar localmente</h2>
1. O ``Node.js`` e o ``npm`` devem estar instalados.

2. Dependências:
```
$ npm install
```

3. Para executar as migrações:
```
$ npm run migrate
```

4. Para iniciar o servidor:
```
$ npm start
```
